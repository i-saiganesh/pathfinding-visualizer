## üí° Overview
This project is a sophisticated visualization tool designed to help users understand pathfinding algorithms. It features a fully interactive grid where you can draw walls, move nodes, and watch algorithms find the shortest path in real-time.

## ‚ú® Key Features
* **Real-time Visualization:** Watch algorithms explore nodes and find the shortest path.
* **Maze Generation:** Automatically generates complex mazes using Recursive Division.
* **Interactive Grid:** Drag & drop start/finish nodes and draw your own walls.
* **4 Major Algorithms:** Includes Dijkstra, A* (Weighted), BFS, and DFS.
